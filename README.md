# Setting my PC

Script to configure the usual programs and development of my PC.

1. Download the "setuplinux.sh" file that is in the project
2. On the terminal, grant the permissions to the file
```
chmod 777 setuplinux.sh
```
3. Then run the file
```
./setuplinux.sh
```
